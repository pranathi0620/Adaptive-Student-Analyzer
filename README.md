# Adaptive Student Performance & Career Readiness Analyzer

## Overview
This project is a Java-based console application that analyzes student academic performance, attendance, and study behavior to generate an explainable career readiness score and internship recommendation.

## Key Features
- Performance trend analysis (Improving / Stable / Declining)
- Weak subject identification
- Attendance impact evaluation
- Career readiness score (out of 100)
- Explainable recommendations

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Java Collections
- VS Code
